Create a web based clone of the classic asteroids arcade game

A welcome screen should display with a grid of 100 numbered levels

Level 1 is unlocked, a level must be completed to unlock the next level

Player has a single life to complete a level

Gameplay increases in difficulty with each level

Each level has a single wave of rocks/enemies

Each level is timed, completion time is the 'high score'

After completing a level or dying, return to the level select screen
